### TextMate und Jekyll

  * [jekyll.tmbundle][3]: This TextMate bundle is fully customized for Octopress, a blog engine based on jekyll. It provides a group of snippets for Octopress Tags, such as codeblock, gist, etc. And it provides several powerfull commands, which can build, launch jekyll or publish blog directly from TextMate.
  * [Jekyll.tmbundle][4]: This bundle provides dead simple support for Jekyll in TextMate. A language file is included which should identify HTML or Markdown files with [YAML Front Matter][9]. A template is included for a blank blog post; create a new file from the Jekyll template. Build your site with ⌘+B
